Usage:

Setup:
```
cd <Directory_to_files>
python -m venv venv
.\venv\Scripts\activate.ps1
pip install -r requirements.txt
```

Starting (After Setup):
```
python Record.py
cd <directory_to_files>
.\venv\Scripts\activate.ps1
python Record.py
```

Buttons:
Start: Starts recording
Stop: Stops Recording
Export: exports to .avi
Pause: Pauses Recording
